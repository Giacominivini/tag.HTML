O que é HTML? O que é HTML?

<html>: Esta é a raiz de um documento HTML. Todos os outros elementos devem ser descendentes dessa tag.

Head

<head>: Este elemento contém metainformações sobre o documento, incluindo o título, scripts e links para folhas de estilo.

Body

<body>: Este é o contêiner para o conteúdo principal de um documento HTML. Geralmente inclui texto, imagens, tabelas, etc.

Header

<header>: Representa um contêiner de conteúdo introdutório ou de navegação. Pode conter alguns elementos de cabeçalho,
mas também outros elementos como um logotipo, um formulário de pesquisa, etc.

Section

<section>: Representa uma seção autônoma de um documento, que poderia ser tabulada ou aparecer em um esboço de um documento.

Div

<div>: É um contêiner que divide o documento em seções. É usado com CSS para estilizar seu conteúdo.

Footer

<footer>: Representa um rodapé para o conteúdo mais próximo de seu ancestral de seção ou o rodapé para o corpo do documento.

H1 até H7

<h4>: Representam seis níveis de títulos de seção, h1 é o mais importante e h6 é o menos importante.

hr

<hr>: Representa uma mudança temática no conteúdo; por exemplo, uma mudança de cena em uma história, ou uma mudança de tópico dentro de uma seção.

p

<p>: Representa um parágrafo de texto.

ul

<ul>: Define uma lista não ordenada (com marcadores).

li

<li>: Define um item de lista, geralmente usado dentro de uma lista (ul ou ol).

br

<br>: Insere uma quebra de linha.
